# Bug Analysis — Complete
Generated: 2026-04-16

## What was built

### analyse.js
Node.js script that performs three-layer analysis on every page/viewport scoring <90%:
1. Region diff analysis — divides diff PNGs into 5 horizontal bands (nav/hero/content/pre-footer/footer), counts red diff pixels, flags bands with >15% density
2. DOM structure diff — uses Playwright to visit both old and new sites, extracts headings/nav links/CTAs/images/section counts, reports missing or changed elements
3. CSS style diff — extracts computed styles (h1, h2, nav, button, body) from both sites, flags mismatches

### build_bug_report.js
Generates a dark-themed HTML report and A3 landscape PDF from bugs.json.
- Summary stats with severity and fix-tag breakdowns
- Per-page issue tables sorted by severity
- Quick Wins section listing all AI-fixable issues
